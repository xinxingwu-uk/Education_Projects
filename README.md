## üß† Educational Philosophy

These projects are designed with the following principles:

- **Learning by building** ‚Äì every concept is reinforced through implementation  
- **Transparency over abstraction** ‚Äì minimal frameworks where possible  
- **Human-centered computing** ‚Äì interaction, perception, and user experience matter  
- **Applied AI** ‚Äì AI as a tool, not a black box  
- **Future-ready skills** ‚Äì aligned with modern industry and research practices

## üß™ Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Interaction**: DOM APIs, animations, audio, camera input
- **AI Concepts**: decision logic, heuristics, game AI, basic perception
- **Tools & Platforms**:
  - Browser-native APIs
  - Node.js (for selected projects)
  - Client-side AI workflows

No heavy frameworks are required for most projects, making them easy to run, modify, and teach.
